In addition to the predefined color names, colors can be specified using RGB, HEX, HSL, or even transparent colors using RGBA or HSLA color values.

HTML RGB and RGBA Colors

An RGB color value represents RED, GREEN, and BLUE light sources. An RGBA color value is an extension of RGB with an Alpha channel (opacity).

rgba(255, 99, 71, 0.2)

rgba(255, 99, 71, 0.8)

rgba(255, 99, 71, 1)

HTML HEX Colors

A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue) hexadecimal integers specify the components of the color.

HTML HSL and HSLA Colors

SL stands for hue, saturation, and lightness. HSLA color values are an extension of HSL with an Alpha channel (opacity).
